# seealso

<code>seealso</code>添加相关的内部或者外部链接到当前页面的底部。

```XML
<seealso>
    <category ref="settings">
        <a href="drum_settings.topic"/>
    </category>
    <category ref="external">
        <a href="https://blog.post.com/drums-are-cool">Why drums are cool</a>
    </category>
</seealso>
```

<note>
需要在c.list注册categories
</note>
